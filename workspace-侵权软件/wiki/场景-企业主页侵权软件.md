# 场景-企业主页侵权软件

## 场景描述

用户在企业主页「经营风险」模块下，查看该企业被工信部通报的侵权软件记录列表。

## 展示入口

- 位置：经营风险 → 公示催告下方（样例内容，需正式确认）
- 标签：带「NEW」标签（显示时长条件未定义 ⚠️）

## 前端字段映射

| 前端展示字段 | 数据表来源 | 说明 |
|-------------|-----------|------|
| 应用名称 | 主表.app_name | - |
| 应用来源 | 主表.app_development_market | - |
| 应用版本 | 主表.app_version | 原PRD写法不一致，应统一为点号格式 |
| 所涉及的问题 | 主表.app_question | 原PRD写法不一致，应统一为点号格式 |
| 发布日期 | 主表.publish_date | 格式 yyyy-MM-dd |

## 数据查询规则

### 通用规则

| 规则项 | 规则内容 |
|--------|---------|
| 有效数据过滤 | data_status = 0 |
| 展示状态过滤 | riskbird_status = 0 |
| 企业关联过滤 | entid > 0 |
| 关联方式 | 通过 entid 关联企业 |

### 个性化规则

| 规则项 | 规则内容 |
|--------|---------|
| 默认排序 | publish_date 降序（DESC） |
| 排序切换 | 支持前端点击切换升序（ASC） |
| 分页 | 默认10条/页（⚠️ PRD未明确，待确认） |
| 时间格式 | yyyy-MM-dd |

## Web 端 UI

### 汇总入口

- 汇总入口展示字段：应用名称、应用来源、应用版本、所涉问题、发布日期
- 支持升降序排序切换

### 列表页

- 展示字段：应用名称、应用来源、应用版本、所涉问题、发布日期
- 筛选项：无
- 详情页：无
- 脱敏规则：无
- 链接规则：无

### 详情弹窗

- 展示字段：应用名称、应用来源、应用版本、所涉问题、发布日期

## 移动端 UI

- 布局方式：卡片式堆叠布局
- 列表页字段：应用名称（置于卡片左上方，不展示字段名）、应用来源、应用版本、所涉问题、发布日期
- 详情页：无
- 筛选项：无

## 四态 UI（⚠️ PRD 未定义，评审建议补充）

| 状态 | 建议文案/样式 |
|------|-------------|
| 加载中 | Skeleton 骨架屏 |
| 请求失败 | 「数据加载失败，请刷新重试」 |
| 筛选无结果 | 不适用（无筛选项） |
| 空数据 | 「暂无侵权软件记录」 |

## 关联系统

### 客诉系统
- 支持用户反馈侵权软件数据问题
- 反馈模式与企业主页「立案信息」反馈模式一致（⚠️ 具体规则未展开）

### 企业报告导出
- 支持维度数据导出
- 导出字段与列表页字段保持一致
- 导出条数上限：⚠️ 未定义（需标注 TBD）
- 导出格式：⚠️ 未定义

### 单维度数据导出
- 支持单维度数据导出
- 导出字段与列表页字段保持一致
- 与企业报告导出的区别：⚠️ 未说明
