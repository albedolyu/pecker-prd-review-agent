# 场景：企业主页纳税人资质模块

## 展示入口

- **菜单路径**：经营信息 → 税务信用评级下方
- **模块名称**：纳税人资质
- **视觉标识**：NEW 标签（消失条件未定义 ⚠️）
- **平台支持**：Web 端、App 端（风鸟端）
- **原型图**：[Figma](https://www.figma.com/design/HCTZuwcQuqvvRirfCgKS5Y/🌟风鸟-原型设计?node-id=482-4578)

## Web 端列表页字段

| 列名 | 说明 |
|------|------|
| 序号 | - |
| 纳税人识别号 | - |
| 纳税人资格类型 | stype 枚举映射，见 [[概念-stype枚举映射]] |
| 主管税务机关 | 文本 |
| 认定有效期起 | 格式：yyyy-MM-dd |
| 认定有效期止 | 格式：yyyy-MM-dd |

## 移动端适配

- **布局方式**：卡片式堆叠布局
- **列表页字段**：纳税人识别号、纳税人资格类型、主管税务机关、认定有效期起、认定有效期止
- **长文本截断规则**：未定义 ⚠️（评审建议补充）

## 交互规则

| 功能项 | 规则说明 |
|--------|---------|
| 空字段 | 字段空值统一显示为"-" |
| 导出功能 | 支持导出当前结果，上限/格式/权限未定义 ⚠️ |

## 四态 UI 规范（PRD 中未完整定义 ⚠️）

| 状态 | 触发条件 | 评审建议展示方式 |
|------|---------|----------------|
| 加载中 | 接口请求中 | 骨架屏/spinner |
| 请求失败 | 接口异常 | 「数据加载失败，请刷新」+ 重试按钮 |
| 无数据 | entid 无对应记录 | 「暂无纳税人资质数据」 |
| 空字段 | 单字段为 null | 显示 - |

## 企业标签

满足条件的企业在搜索结果页和企业主页头部展示对应纳税人资质标签：
- tagid 99999997 / 99999998（业务含义未在 PRD 中说明 ⚠️）
- 标签颜色：蓝色，与【企业】【大型】等标签相同样式
- 数据来源：[[约束-t_ds_aggre_tags表结构]]

## 客诉反馈

- 提供客诉反馈入口，样式与税务信用评级（无案号反馈样式）一致
- 具体规范依赖外部文档，未在 PRD 内内联定义 ⚠️

## 已知盲区

- NEW 标签展示时长/消失条件未定义
- 空数据态（模块隐藏 or 占位文案）未定义
- 加载失败态展示规则未定义
- 导出上限/格式/权限未定义
- tagid 99999997/99999998 的业务含义未明确
- 「状态」字段（tstatus）去留无决策记录
- sdate NULL 值排序行为未定义
