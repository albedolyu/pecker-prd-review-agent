# 搜索优化 PRD

## 1. 背景
风鸟 APP 的搜索功能需要优化，当前搜索结果不够精准。

## 2. 目标
提升搜索准确率到 90%。

## 3. 功能需求
### 3.1 搜索逻辑
支持企业名称、法人、统一社会信用代码的模糊搜索。

### 3.2 筛选条件
按地区、行业、注册资本范围筛选。

## 4. 字段映射
| PRD字段 | 数据表 | 字段名 |
|---------|--------|--------|
| 企业名称 | company_info | company_name |
| 法人 | company_info | legal_person |
| 注册资本 | company_info | registered_capital |
