# 场景-产品召回列表页

## 用户场景

用户在风鸟平台「经营风险 → 产品召回」模块，查看某企业历史产品召回记录。

## Web 端展示字段

| 列名 | 对应字段 | 说明 |
|------|---------|------|
| 召回产品 | recall_product | - |
| 召回数量 | recall_num | - |
| 召回原因 | recall_reason | 超2行展开收起 |
| 监管部门 | supervision_department | - |
| 发布日期 | rdate | 默认 DESC 排序，可切换 ASC |
| 内容 | - | 触发详情弹窗的文字链接 |

## App 端展示字段

召回产品、召回数量、召回原因、发布日期（详情入口未定义，见 R-C01）

## 过滤条件

- `entid` = 当前企业
- `data_status = 0`
- `riskbird_status = 0`（⚠️ 字段来源待确认，见[[决策-产品召回评审发现]]）
