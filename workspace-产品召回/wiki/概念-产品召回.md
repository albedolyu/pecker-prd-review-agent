# 概念-产品召回

## 定义

产品召回是指企业因产品存在缺陷或安全隐患，由监管部门或企业自发发布公告，要求收回已流通产品的行为。属于企业经营风险维度的重要数据类型。

## 业务背景

- 竞品企查查已有该维度数据
- 数据来源：智慧星光、as（内部缩写，全称待确认）
- 在风鸟平台「经营风险」模块下新增，位于「公示催告」下方

## 核心字段（来自 ds_risk_recall）

| 字段 | 含义 |
|------|------|
| entid | 企业ID，关联主体 |
| rdate | 发布日期 |
| recall_product | 召回产品名称 |
| recall_num | 召回数量 |
| recall_reason | 召回原因 |
| content | 公告正文 |
| source | 信息来源 |
| title | 公告标题 |
| data_status | 数据状态，0=有效 |
