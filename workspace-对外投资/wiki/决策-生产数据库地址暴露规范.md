# 决策-生产数据库地址暴露规范

## 问题描述

PRD「数据维度-对外投资增加间接投资」v1.2 的 §2.0 数据源定义中，明文写入了生产数据库 IP 地址和端口：

- `106.75.95.213:12571`（MySQL，dataplatform）
- `106.75.87.237:5432`（PostgreSQL，ds_company）

## 当前状态

⚠️ **待验证**：团队尚未形成明文文档规范，本条目记录该问题，等待 PRD 作者确认处理方式。

## 建议处理方式

将生产连接信息替换为配置项占位符，例如：

| 配置项 | 说明 |
|--------|------|
| `${DS_MYSQL_HOST}:${DS_MYSQL_PORT}` | MySQL 连接地址（dataplatform） |
| `${DS_PSQL_HOST}:${DS_PSQL_PORT}` | PostgreSQL 连接地址（ds_company） |

实际值通过配置中心（如 Apollo/Nacos）或环境变量管理，不写入 PRD/代码仓库。
