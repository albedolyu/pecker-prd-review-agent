# 决策：移动端保留认缴出资额（端差异）

## 决策结论

移动端直接投资页面**保留认缴出资额字段**，作为与 Web 端的端差异设计：

- 上市公司：展示**持股量**
- 非上市公司：展示**认缴出资额**

Web 端直接投资列表按 FR-02 要求，**剔除认缴出资额**。

## 决策背景

- PRD v1.2 FR-02 明确「直接投资列表：剔除认缴出资额」，指向 Web 端
- PRD 4.5节移动端适配中保留了认缴出资额字段（「如果非上市公司 展示 认缴出资额」）
- 评审确认：移动端的保留属于**有意的端差异设计**，非遗漏

## 待补充 FR

需在功能需求列表中新增：
