## 概念页（3）

- [[概念-对外投资]] — 对外投资是指企业作为股东，向其他企业进行的股权投资行为。分为直接投资（ds_company_discover_invest）和间接投资（ds_invest_alpha）两类。
- [[概念-持股合计]] — 持股合计是指某企业在被投资企业中的最终持股比例，直接投资来源 conprop，间接投资来源 invest_ratio。
- [[概念-间接投资持股计算]] — 间接投资持股比例的穿透计算逻辑（占位页，待补充）。

## 场景页（5）

- [[场景-对外投资列表浏览与筛选]] — 用户在企业详情页查看该企业的对外投资情况，分为「直接投资」和「间接投资」两个 Tab。
- [[场景-对外投资筛选逻辑]] — 对外投资筛选条件与 WHERE 条件映射（占位页，待补充）。
- [[场景-投资详情弹窗]] — 点击持股合计 icon 弹出投资详情（股权链路）弹窗（占位页，待补充）。
- [[场景-直接投资列表页]] — 用户在企业详情页查看该企业的直接对外投资记录，支持筛选、搜索和导出。
- [[场景-间接投资列表页]] — 用户在企业详情页查看该企业通过股权穿透后的间接投资企业列表，支持筛选、搜索和导出。

## 约束页（4）

- [[约束-ds_company_discover_invest表结构]] — 直接投资物理表（MySQL），记录企业直接持股关系，持股比例字段为 conprop（0-100）。
- [[约束-ds_invest_alpha表结构]] — 间接投资穿透表（PostgreSQL），记录企业通过多层股权穿透后的间接持股关系，持股比例字段为 invest_ratio。
- [[约束-t_ds_company_basic表结构]] — 企业基本信息表（MySQL），被对外投资模块通过 entid_inv / invest_id JOIN 使用，含行业分类（nic_id 17版、nic_id_bak 13版）。
- [[约束-对外投资DDL]] — 对外投资三张物理表综合 DDL 参考。

## 决策页（4）

- [[决策-移动端保留认缴出资额]] — 移动端直接投资页面保留认缴出资额字段，与 Web 端差异化设计（⚠️ 是否仍有效待确认）。
- [[决策-股权链展示上限]] — Web 端股权链最多展示 10 条链路（⚠️ 待确定项，降级策略未明确）。
- [[决策-生产数据库地址暴露规范]] — PRD 中不应明文写入生产数据库 IP/端口，应替换为环境变量占位符。（来源：第二轮人工评审 R-002）
