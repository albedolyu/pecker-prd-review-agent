# 场景-对外投资列表浏览与筛选

## 用户场景描述

用户在企业详情页查看该企业的对外投资情况，分为「直接投资」和「间接投资」两个 Tab。

## 直接投资列表

- 数据来源：`ds_company_discover_invest` 表，通过 `entid` 关联
- 展示字段：被投资企业、法人代表、注册资本、成立日期、地区、行业、直接投资比例、持股合计
- 筛选维度：持股合计（枚举+自定义）、经营状态、所属地区、所属行业（二级）、搜索框
- 默认排序：经营状态在营优先 → 持股合计倒序 → 注册资本倒序
- 分页：默认20条，支持切换50/100条（TBD）

## 间接投资列表

- 数据来源：`ds_invest_alpha` 表，剔除直接投资部分，通过 `entid` 关联
- Tab 标题旁有 NEW 标识
- 展示字段：被投资企业、法人代表、注册资本、成立日期、地区、行业、持股合计
- 筛选维度：持股合计（枚举+自定义）、经营状态、所属地区、所属行业（二级）
- 默认排序：与直接投资一致

## 筛选规则

| 筛选维度 | 枚举值 |
|---------|--------|
| 持股合计 | 不足5% / 5%及以上 / 25%及以上 / 33.33%及以上 / 50%及以上 / 66.66%及以上 / 100% / 自定义 |
| 经营状态 | 按企业经营状态筛选 |
| 所属地区 | 省级联动，二级联动 |
| 所属行业 | 二级行业，展示一级/二级隶属关系 |

## 四态 UI 规范

| 状态 | 直接投资/间接投资列表 |
|------|-----------------|
| 加载中 | 表格行显示骨架屏 |
| 加载失败 | 显示「数据加载失败，请刷新重试」，提供刷新按钮 |
| 筛选无结果 | 显示「暂无符合条件的数据」，保留筛选区可操作 |
| 空数据 | 显示「暂无对外投资数据」 |
