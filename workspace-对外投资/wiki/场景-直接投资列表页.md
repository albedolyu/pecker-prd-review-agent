# 场景-直接投资列表页

## 场景描述

用户在企业详情页查看该企业的直接对外投资记录，支持筛选和导出。

## 展示字段

| 字段 | 数据来源 |
|------|---------|
| 被投资企业 | `t_ds_company_basic.entname`（通过 entid_inv 关联） |
| 法人代表 | `t_ds_company_basic.oper_name` |
| 注册资本 | `t_ds_company_basic.regcap` |
| 成立日期 | `t_ds_company_basic.esdate` |
| 地区 | `t_ds_company_basic.prov` |
| 行业 | `t_ds_company_basic.nic_id`（待确认字段名，非nacaoid） |
| 直接投资 | `ds_company_discover_invest.invest_ratio` |
| 持股合计 | `ds_company_discover_invest.invest_ratio`（直接投资时与直接投资相同） |

## 筛选维度

- 持股合计（范围：0~25%、25~50%、50~100%）
- 经营状态（`entstatus_cn`）
- 所属地区
- 所属行业（一级/二级，字段待确认）

## 移动端特殊规则

- 上市公司展示「持股量」，非上市公司展示「认缴出资额」
- 判断上市的字段来源**待补充** ⚠️

## 已知问题

- R-005：行业筛选字段 nacaoid 使用有误，应为 nic_id 或 nic_id_bak
- R-007：上市/非上市判断字段未定义
- R-012：默认排序规则未定义
