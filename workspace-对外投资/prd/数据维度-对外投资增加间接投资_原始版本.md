## 执行摘要

本需求在现有「对外投资」模块基础上新增「间接投资」标签页，将原有对外投资改为「直接投资」，并通过股权穿透数据（ds_invest_alpha）计算间接持股关系。主要改动：
1. 对外投资拆分为直接投资和间接投资两个 tab；
2. 直接投资剔除认缴出资额，增加持股合计字段；
3. 间接投资为新增 tab，数据来自 ds_invest_alpha 表（剔除直接投资部分）；
4. 两个 tab 均支持筛选、搜索、分页、导出；
5. 持股合计点击弹出投资详情（股权链）。

**筛选条件返回规则**：

- 空值处理：数据库中空值的记录统一归类为"其他"
- 数量统计：每个筛选项需返回该维度下的记录数量
- 调用时机：仅当请求参数指定需要筛选条件时才返回（避免不必要的聚合查询）
- 按需返回：非首次加载或用户点击筛选时才请求筛选条件数据

**关于二级行业删选的特殊说明**：

如果对外投资有2家企业，1家企业只有一级行业计算机、另外1家企业有二级行业，为计算机-软件业。所属行业的筛选希望为: 一级行业只展示计算机，二级行业展示计算机（计算机从1级行业继承过来的）和软件业

**二级行业筛选构建逻辑（伪代码）：**

```python
# 从当前列表数据中聚合行业筛选项
industry_options = []

for company in current_list:
    level1 = company.nic_id_cn_level1  # 一级行业，如「计算机」
    level2 = company.nic_id_cn_level2  # 二级行业，如「软件业」（可为空）

    # 一级行业始终作为独立选项
    add_if_not_exists(industry_options, level1)

    if level2:
        # 有二级行业则加入二级行业
        add_if_not_exists(industry_options, level2)
    else:
        # 无二级行业，将一级行业作为二级行业继承展示
        add_if_not_exists(industry_options, level1)

# 最终展示：一级仅展示去重后的一级行业名
# 二级展示所有二级行业 + 无二级行业企业的一级行业（继承）
```
