落地规则

  1. 所有锚点联动必须双向：点评论 → 原文高亮；点原文 → 评论高亮。
  2. 高亮时长固定 300ms，禁止更久——久了成"选中态"，破坏了"一瞥反馈"的意图。
  3. 禁用 element.scrollIntoView()（项目约束），使用容器的 scrollTo({ top, behavior: 'smooth' })。
  4. `prefers-reduced-motion: reduce` 时，关闭 smooth-scroll 与 flash，仅保留 focus-ring。
  5. 键盘 Enter 触发同样联动，focus-ring 留到下次按键 / blur 才淡出。
  6. 同时激活多条评论时，只 flash 最近一条——不要连锁动画。
  7. Phase 2 期间 drawer 不显示；这套规则仅 Phase 3 适用。
